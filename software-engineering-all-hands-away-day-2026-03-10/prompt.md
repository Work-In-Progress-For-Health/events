Create a software engineering all hands away day.

Audience is NHS Wales staff:

- software engineers
- software developers
- UI/UX designers
- data analysts
- test automation specialists
- information security staff

Read CLAUDE.md to understand 8 event sessions structured as a full-day away day plan.

- Each session follows the 15-minute talk + 15-minute workshop format
- Each session builds on previous sessions (from vision through strategy, culture, principles, moonshots, execution, measurement, and professional growth)
- Each session surfaces the fears and levers for each topic.

Tone is inspirational, encouraging, hopeful.

The arc of the day bookends with the same three levers -- involvement, empowerment, encouragement -- in Sessions 1 and 8.

- Create a section with workshopping tutorial ideas, suggestions, insights, ideads, etc.
